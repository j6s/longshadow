LONGSHADOW

a simple LESS mixin

what is longshadow?

longshadow.less is a simple LESS mixin, that lets you create awesome longshadow headings and icons unsing the power of CSS3 - no images required. It builds upon the article Create a Flat, Long Shadow Icon with CSS3 by Matt Lambert


how does it work?

the heart of longshadow.less is the .longshadow mixin. It is really easy to use:


h1.shadow {
    // creates a #555 colored shadow, 50px long
    .longshadow(50,#555);
}

.awesomeIcon {
    // creates a #f19103 - colored icon
    .longshadowIcon(@color: #f19103);
}

but wait - there's more

the .longshadowIcon mixin can be adjusted to your need. Simply pass one these variables with it:

  • @color - the main / background color of the icon
  • @secondColor - the accent color of the icon (used for the border and the text)
  • @depth - the length of the shadow. Adjust this if you see the end of it
  • @padding - the padding used for the icon
  • @fontSize - i think you can guess what this does, can't you?
  • @borderSize - thickness of the border


before usage

before using this you have to be clear about one thing:
longshadow.css creates imense amounts of css output. For this page the CSS file is about 6kb, most of it for the big shadow in the heading and the small subheadings.
